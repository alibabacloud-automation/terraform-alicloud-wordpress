Terraform模块，部署基于阿里云ECS实例和云数据库Mysql的wordpress

terraform-alicloud-wordpress
=====================================================================

## 用法

要运行此示例，需要执行:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

注意，这个例子将创建花费金钱的资源。不需要这些资源时运行“terraform destroy”
